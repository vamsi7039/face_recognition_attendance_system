
# Face Recognition Attendance System

This project implements a face recognition attendance system using CNN and transfer learning with TensorFlow/Keras.

## Setup
1. Create a Python 3.13.9 environment.
2. Install required packages: tensorflow, opencv-python, numpy, matplotlib.
3. Run `train.py` to train or fine-tune the model.
4. Run `attendance.py` to detect faces and mark attendance.
